viewpagerindicator-aar
======================

An aar maven artifact for ViewPagerIndicator.

###Usage

    repositories {
        ...

        maven {
            url "https://github.com/hdodenhof/viewpagerindicator-aar/raw/master"
        }
    }

    dependencies {
        ...
        compile 'com.viewpagerindicator:viewpagerindicator:2.4.2'
    }
